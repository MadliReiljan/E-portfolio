Mobile App Example PROJECT

This project is a furniture shopping platform designed to let users browse and post furniture listings. The app was built using React, Javascript, Appwrite and Expo Go. The app lets users sign up, log in, explore listings, search for items, save to favorites and upload their own furniture images to share with the community.

What I Learned

Having built a similar app before (Aora), I came into this project with a solid foundation in mobile app development. This time, I focused on strengthening my understanding of app structure and working with react. Using Appwrite again for authentication and session management felt familiar, because it is a straightforward and reliable for handling user login, registration and logout functions.

One of the biggest challanges was trying to set up Google login. The main issue was with Expo Go, which required a different approach to make it work properly. Despite trying various methods, I couldn't get Google login fully functional, so I haven't included it into the project. However, this process taught me a lot about debugging - especially in identifying common issues like import errors and missing symbols. I have improved my ability to diagnose errors and understand their causes, which has strengthened my troubleshooting skills overall.
